About this application

Listed in this page are resources I used to build this application

Inspiration and Credits

For this application I will credit @DevDreamerCode for inspiring me to build this clone. You can visit their linked IMDB clone here: http://bit.ly/2KmIpPQ

TypeScript

TypeScript, the superset langauge of JavaScript, is used by the Angular framework (as well as many others) to enforce strict datatypes when necessary.

Angular

Angular 13 was the backbone of this project as it offered many great features to leverage.

RxJS

RxJS was used for simplifying asynchronous API/http requests.

Google Firebase

Firebase helped with providing a realtime database and user authentication for managing their favorite movies.

HTML5

HTML5 was used (clearly) to organize the page structure.

CSS3

Vanilla CSS was used for this project and its animations. I did not use any third-party libraries.

Font-Awesome

Font Awesome is what I used for my icons. They offer a rich variety of free icons to choose from.
